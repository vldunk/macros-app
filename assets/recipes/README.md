# Recipe Images

All recipe photos live in this folder:

```
assets/recipes/
```

Each filename must match the recipe slug:

```
{slug}.webp
```

Example:

```
assets/recipes/greek-yogurt-berries.webp
```

The catalog stores the same path in `image_url`:

```js
image_url: "assets/recipes/greek-yogurt-berries.webp"
```
